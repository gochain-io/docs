# Public GoChain Network

### Network RPC URLs

When interacting with GoChain, the easiest way is to use our hosted JSON-RPC API's. The base URLs
are:

* GoChain TestNet: https://testnet-rpc.gochain.io
* GoChain MainNet: https://rpc.gochain.io
